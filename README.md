- **Объяснение наличия интерфейса lo:**
  - Интерфейс `lo` (loopback) — это виртуальный сетевой интерфейс, который используется системой для связи процессов внутри одного компьютера. Он всегда имеет IP-адрес 127.0.0.1 и необходим для работы многих системных служб и приложений, которые общаются между собой через сетевые сокеты, но находятся на одной машине.

- IP-адрес, полученный от DHCP-сервера:
  - ![IP адрес](screenshots/part3_ip.png)
  - *На скриншоте показан IP-адрес, полученный от DHCP-сервера*

- **Расшифровка DHCP:**
  - **DHCP** (Dynamic Host Configuration Protocol) — протокол динамической настройки узла. Это сетевой протокол, который позволяет устройствам автоматически получать IP-адрес и другие параметры сети (маску подсети, шлюз по умолчанию, DNS-серверы) от DHCP-сервера. Благодаря DHCP администраторам не нужно вручную настраивать каждое устройство в сети.
